                                豆包(火山引擎) 语音接入教程

                                目前火山引擎对于语音合成(TTS)有“旧版语音控制台”和“新版大模型控制台(方舟)”两种不同的接入方式。本APP目前使用的是标准长文本/短文本在线语音合成（非端到端大模型）。

                                步骤 1：开通服务并获取 App ID
                                1. 访问火山引擎官网，进入 语音技术控制台 -> 应用管理
                                2. 点击“创建应用”，在应用详情页可找到 AppID。
                                3. 在“语音合成”能力中，选择并开通你需要的音色服务（例如：在线语音合成）。注意：必须开通服务否则会报鉴权错误！
                                步骤 2：获取 Access Token (API Key)
                                1. 不要使用旧版的 Authorization Token 逻辑。
                                2. 进入右上角个人头像 -> API Key 管理。
                                3. 新建一个 API Key，将其复制填入设置页面的 Access Token 框中。
                                步骤 3：集群 (Cluster) 选择

                                通常填写 volcano_tts。如果你使用的是特定的复刻音色，请在官方文档中查找你所用音色对应的特定 Cluster 名称填入。
